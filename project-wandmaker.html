Wandmaker's Atelier

Process

The aim of this project was to build an interactive wand-crafting simulation in Java, where users make sequential material and design choices through a mouse-driven GUI, with each decision dynamically shaping the wand's final power and appearance. The brief required blending solid software design patterns with generative visuals to create an experience that felt immersive and polished rather than purely functional.

I built the full simulation independently, implementing the Decorator design pattern to layer each user choice so that material and customization decisions accumulated meaningfully into a calculated final outcome. To bring the world to life, I generated complex animated visuals using recursion and Perlin noise, producing fractal patterns, branching effects, and organic particle movement. I also programmed a day and night cycle with smooth transitions and designed four distinct spell effects, each paired with synthesized audio feedback. The most technically demanding part of the project was generating fractal and Perlin noise visuals that felt meaningfully connected to the wand-crafting context rather than serving as decorative afterthoughts. Getting the recursive algorithms to produce organic, on-theme results required significant experimentation, adjusting parameters, testing outputs, and iterating until the visuals read as a natural extension of the world rather than a technical demo. I also conducted iterative peer testing throughout development, using that feedback to refine hover states and contextual tooltips so users were guided naturally through each stage.

The result is a fully playable, visually polished interactive simulation that demonstrates both software architecture knowledge and a strong sense of atmospheric design, with the technical and creative decisions reinforcing each other throughout. This project reflects my ability to apply structured programming patterns to generative, visually driven systems, and to treat technical implementation and aesthetic intent as part of the same problem rather than separate concerns.
